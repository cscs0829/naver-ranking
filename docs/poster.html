네이버 쇼핑 순위 검색기
개발 기간: 2024.01 ~ 2024.06
참여 인원: 1명 (개인 프로젝트)
담당 역할: 풀스택 개발
주요 기능
  • 네이버 쇼핑에서 특정 상품의 정확한 순위 검색
  • 키워드 트렌드 분석 및 다중 키워드 비교
  • 정기적인 자동 검색 스케줄링 (30분~1개월)
  • 실시간 알림 및 대시보드를 통한 모니터링
  • 엑셀 파일로 검색 결과 내보내기
주요 기술
  • 네이버 쇼핑 API를 통한 최대 1,000개 상품 검색
  • 네이버 데이터랩 API 활용 및 트렌드 시각화
  • Supabase PostgreSQL을 통한 데이터 관리
  • Node.js 스크립트를 활용한 자동화 시스템
  • 실시간 알림 및 대시보드 통계 제공
